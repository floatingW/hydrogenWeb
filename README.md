# hydrogenWeb
## 简介
hydrogen，即氢气。hydrogenWeb是一个轻量的web server。使用C++/CMake/GTest，无其他依赖。
## roadmap
1. 迭代服务器（以简单的echo服务程序为例）
2. 并发echo服务器（基于进程、线程、IO多路复用的实现）
3. 并发Web服务器
   1. 初步主体框架 - 示意图
   2. 初步主体框架 - 各模块函数、库的选用
   3. 初步主体框架的基本实现
